## About the Blog

This blog serves as a technical knowledge base documenting my work in cybersecurity. It focuses on practical, hands-on content rather than general commentary or personal updates.

The primary goal is to maintain a public archive of security research, walkthroughs, and experiments that are accurate, reproducible, and useful to other practitioners.

---

## Content Includes

* **CTF Writeups**
  Step-by-step breakdowns of challenges with reasoning, tooling, and exploitation logic.

* **Walkthroughs**
  Structured walkthroughs for platforms such as PortSwigger Web Security Academy, TryHackMe, and similar environments.

* **Security Tutorials**
  Focused explanations of specific techniques, vulnerabilities, and attack patterns.

* **Research Notes & Experiments**
  Hands-on exploration of modern security topics, including LLM security and web vulnerabilities.

---

## Blog Structure

The blog is organized to make technical content easy to navigate and reference over time:

* **Home**
  Displays the latest posts, including CTF writeups, walkthroughs, and security research articles.

* **Categories**
  Groups posts by domain (e.g., Web Security, LLM Security, Walkthroughs, CTFs) to help readers focus on specific areas.

* **Tags**
  Provides fine-grained filtering across topics, tools, and techniques used throughout the blog.

* **Timeline**
  Shows posts chronologically, acting as a public record of ongoing security work and learning progression.

* **About**
  Describes the purpose of the blog, technical focus areas, and scope of content.

* **Series Pages**
  Long-running challenges (such as *Advent of Cyber 2025*) are organized into dedicated sections with an index page linking all related walkthroughs.
